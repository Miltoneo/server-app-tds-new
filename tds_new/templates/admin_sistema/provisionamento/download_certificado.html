Próximos Passos
  1. Extraia o ZIP baixado
  2. Gere a imagem NVS da partição certs: 
    python $IDF_PATH/components/nvs_flash/nvs_partition_generator/nvs_partition_gen.py \
    generate certs.csv certs.bin 0x6000
  3. Grave a imagem NVS no dispositivo: 
    esptool.py --port /dev/ttyUSB0 write_flash 0x3FC000 certs.bin
  4. Grave também a partição setup com os dados de fábrica (device_id, modelo, MQTT etc.)
  5. Ligue o dispositivo e verifique a conexão ao broker MQTT via telemetry/{{ certificado.device_id|default:certificado.mac_address }}-<LAST4MAC>/data
